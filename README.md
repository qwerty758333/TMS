# Task Management System (TMS)

A full-stack web application for creating, managing, and tracking tasks collaboratively in real-time.

## 🎯 Project Overview

This is an group project for INTE 21323. The application helps teams organize, assign, and track tasks efficiently.

## 🛠️ Tech Stack

### Frontend
- React 18+
- React Router
- Axios
- Socket.io-client
- CSS3 + Tailwind

### Backend
- Node.js
- Express.js
- Sequelize ORM
- JWT Authentication
- Socket.io

### Database
- **MySQL 8.0+**
- Port: 3306

### DevOps
- Docker & Docker Compose
- GitHub
- AWS/Azure/GCP (for deployment)
